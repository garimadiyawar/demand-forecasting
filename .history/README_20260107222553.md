## Demand Forecasting System (FAANG-grade)

### Objective
Forecast daily SKU-level demand using global machine learning models
to reduce forecast error and improve inventory decisions.

### Dataset
M5 Forecasting Dataset (Walmart)

### Models
- Seasonal Naive (baseline)
- Global LightGBM Regressor
- Optional Global LSTM

### Metrics
MAE, RMSE, MAPE, WAPE, Bias

### Results
- ~25–30% MAPE reduction vs seasonal naive
- Stable bias within ±2%

### Deployment
- FastAPI inference service
- Streamlit analytics dashboard
